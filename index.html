Generic Java Agent Loads a thread class implementation into a running JVM

A generic Java Agent, with a utility using the Attach API which can connect to a running JVM process and launch the agent. Alternatively the standard mechanism for specifying a Java Agent can be used at startup. The agent will start a configurable, specific thread instance inside the running JVM.

  

  java-agent-bootstrap-0.1/inject  -javaagent=java-agent-bootstrap-0.1.jar=libs=/full/path/to/libs,thread=org.example.MyThread
